Build a complete marketing website for a company called Graylock Digital.

Graylock Digital is a subscription-based website company that builds and 
maintains modern, lead-focused websites for small service businesses 
(accountants, therapists, contractors, and similar local professionals).

---

TECH STACK
- Framework: Next.js 14 (App Router)
- Styling: Tailwind CSS
- Animations: Framer Motion
- Forms: Formspree (use a placeholder action URL, I will replace it)
- Fonts: Use Google Fonts — pair "Barlow Condensed" (700 weight) for 
  all headlines and display text with "Barlow" (400/600 weight) for 
  all body text and UI elements. These match the brand's logo typography.
- Deployment ready: Vercel-compatible

---

EXACT BRAND COLORS (use these precisely, do not substitute)
Define these as CSS variables in globals.css and use them everywhere:

--color-charcoal: #0F1117;       /* Hero backgrounds, navbar */
--color-navy: #1A1F2E;           /* Cards, pricing panels, footer */
--color-gunmetal: #2C3140;       /* Borders, dividers, muted elements */
--color-stone: #8A8F9E;          /* Body text on dark, captions, subtext */
--color-offwhite: #F2F3F5;       /* Light sections, headline text on dark */

Tailwind config: extend the theme to include these as custom colors so 
they can be used as Tailwind classes (e.g. bg-charcoal, text-orange, etc.)

BRAND & DESIGN DIRECTION
- Use the exact color palette above. No other accent colors.
- The navbar and hero section use --color-charcoal (#0F1117) as background.
- Cards, pricing panels, and the footer use --color-navy (#1A1F2E).
- Light content sections (How It Works, FAQ preview, About) use 
  --color-offwhite (#F2F3F5) as background with --color-charcoal text.
- The final CTA section uses --color-orange (#E8631A) as background 
  with white text and a --color-charcoal button. Use this treatment 
  ONLY for the final CTA — not for any other section.
- All CTA buttons use --color-orange background with white text.
  Outline button variant: --color-orange border and text, transparent bg.
- Body text on dark backgrounds uses --color-stone (#8A8F9E).
- Headline text on dark backgrounds uses --color-offwhite (#F2F3F5).
- Borders, dividers, and subtle separators use --color-gunmetal (#2C3140).
- Design tone: Refined, modern, confident. Feels like a premium local 
  service firm. Bold typography-driven layouts. NOT startup-flashy.
- Section rhythm: alternate dark/light/dark/light as you scroll:
  Hero (charcoal) → Problem (offwhite) → Solution (navy) → 
  How It Works (offwhite) → Testimonials (charcoal) → 
  Work Examples (navy) → Pricing (charcoal) → FAQ (offwhite) → 
  Trust block (navy) → Final CTA (orange) → Footer (charcoal)
- NO generic purple gradients. NO stock photo hero images.
- All sections should have smooth scroll-reveal animations using 
  Framer Motion (fade up on enter, 0.5s duration, 20px offset).
